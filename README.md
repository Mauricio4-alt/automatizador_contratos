# Automatizador de contratos
## Objetivo General
Este es un proyecto sencillo que tiene como objetivo desarrollar una aplicación de escritorio
que tenga como objetivo la elaboración y automatización de contratos formales para que las 
empresas puedan incorporar nuevo personal a su fuerza de trabajo


## Tecnologías

1. **Python** (lenguaje base)
2. **Tkinter** para la elaboración de la interfaz gráfica (libreria)
3. **python-docx**  para la elaboracion de archivos word (libreria)
3. **Pandas** para manejo de datos tabulares (libreria) util para extraer datos de excel y otros formatos
4. **Path** para manejo de rutas de archivo (libreria)


## Instalación 

1. para instalar este proyecto debe tener python instalado
2. use el comando: **pip install --no-cache-dir -r  requirements.txt**


## ¿Cómo usarlo?

El manejo de este proyecto es simple una vez instalado haga doble click en el documento lectura_excel.pyw

una vez dentro debe tomar en cuenta lo siguiente 

### el documento excel debe contener:
Una tabla con los siguientes items

1. Nombre
2. Documento
3. Cargo

deben estar escritos tal cual como indica este documento con la primera letra en mayúscula y en minúcula las demás
para que no haya errores de ejecución.

Una vez hecho eso use la interfaz para insertar el documento excel con los datos requeridos, puede generar sus contratos
los contratos se cargaran en la misma carpeta donde se encuentra el ejecutable llamada contratos y puede visualizarlos
